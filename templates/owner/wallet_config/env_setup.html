Setup Instructions
Two Configuration Options

You can configure payment providers in two ways:

Option 1: Database (Recommended)
  • API keys encrypted at rest
  • Changes take effect immediately
  • No restart required
  • Owner dashboard interface
Configure in Database
Option 2: Environment Variables
  • Highest security
  • No database storage
  • Requires app restart
  • Manual file editing
See template below
Steps to Configure via .env:
  1. Locate your .env file

    Find the .env file in your project root directory

  2. Get API keys from providers

    Flutterwave Dashboard | Paystack Dashboard

  3. Copy the template below

    Add the relevant configuration to your .env file

  4. Restart the application

    Environment variables require an application restart to take effect

Security Warning

Never commit your .env file to version control. Add .env to your .gitignore file.

Provider-Specific Instructions
Flutterwave
  1. Sign up at flutterwave.com
  2. Go to Settings > API Keys
  3. Copy your Secret Key, Public Key, and Encryption Key
  4. Set FLUTTERWAVE_SANDBOX=true for testing
  5. Add your webhook URL in Flutterwave dashboard:
    {{ request.url_root }}webhooks/flutterwave
Paystack
  1. Sign up at paystack.com
  2. Go to Settings > API Keys & Webhooks
  3. Copy your Secret Key and Public Key
  4. Set PAYSTACK_SANDBOX=true for testing
  5. Add your webhook URL in Paystack dashboard:
    {{ request.url_root }}webhooks/paystack
MTN Mobile Money
  1. Contact MTN Mobile Money team for API access
  2. Get your API Key, API User, and API Secret
  3. Set MTN_MOMO_SANDBOX=true for testing
  4. Configure callback URL:
    {{ request.url_root }}webhooks/mtn-momo
